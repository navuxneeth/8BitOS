# 8BitOS

Retro-inspired single-page desktop experience built with plain HTML, CSS, and JavaScript.

## Features
- Bootloader-style preloader with progress log and sound blips.
- Desktop with draggable retro windows and scanline overlay.
- Start menu and desktop icons with hover/active audio feedback.
- Theme switcher (classic, neon, sunset).
- Apps: System Info, Notepad, MS Paint-style canvas (save/clear), Gallery folder viewer with lightbox, Calculator, Music player (demo tone).
- Responsive layout tweaks for mobile widths.
- Custom pixel cursor and hover effects.

## Usage
Open `index.html` in a modern browser. Audio requires user interaction in some browsers; click an icon or Start button to unlock sound.

## Notes
- Gallery images use placeholder remote assets; swap URLs to serve local content if desired.
- Canvas save triggers a PNG download in supported browsers.
